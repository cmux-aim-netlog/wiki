# `piki/wiki/db.py`

이 모듈은 piki 위키의 내부 상태와 메타데이터를 관리하는 데이터베이스 또는 인덱스 역할을 합니다.

주요 기능은 소스 코드 파일과 위키 페이지 간의 매핑, 심볼 위치, 페이지 간의 링크 관계 등 위키의 일관성과 효율적인 업데이트를 위해 필요한 정보를 저장하고 조회하는 것입니다. 예를 들어, 특정 소스 파일이 변경되었을 때 어떤 위키 페이지를 업데이트해야 하는지 신속하게 파악하는 데 사용됩니다.
